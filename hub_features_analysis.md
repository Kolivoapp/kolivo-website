# KOLIVO HUB - Analyse des Fonctionnalités

## Description Générale
KOLIVO HUB est une **application de gestion de colocation** (shared living / roommate management). C'est une plateforme B2C destinée aux colocataires pour gérer leur vie commune avec bienveillance.

## Fonctionnalités Principales

### 1. Finances Partagées (Expenses)
- Gestion des dépenses communes
- Répartition équitable des coûts
- Suivi des balances entre colocataires
- Rappels de paiement

### 2. Tâches & Corvées (Tasks)
- Attribution des tâches ménagères
- Rotation intelligente
- Respect du rythme de chacun

### 3. Calendrier Partagé (Calendar)
- Événements communs
- Vacances et absences
- Visibilité pour tous les membres

### 4. Épicerie Intelligente (Grocery)
- Liste de courses collaborative
- Détection d'allergènes
- Partage en temps réel

### 5. Inventaire (Inventory)
- Suivi des biens communs et personnels
- Photos des objets
- Gestion des équipements partagés

### 6. Ententes / Accords (Agreements)
- Création d'ententes de colocation
- Signature électronique
- Règles de vie commune

### 7. Documents
- Stockage de documents partagés
- Contrats, factures, etc.

### 8. Annonces (Announcements)
- Communication interne
- Messages importants

### 9. Règles (Rules)
- Règles de la maison
- Politiques internes

### 10. Chat
- Messagerie entre colocataires
- Communication en temps réel

### 11. Maintenance
- Signalement de problèmes
- Suivi des réparations

### 12. Sondages (Polls)
- Votes sur les décisions communes
- Démocratie participative

### 13. Paiements (Payments)
- Intégration Stripe
- Paiements entre colocataires

## Valeurs de la Plateforme
- **Bienveillance** : Zéro culpabilisation, communication non violente
- **Confiance** : Données privées et sécurisées
- **Équité** : Répartition juste qui respecte les situations de chacun

## Contexte Technique
- Application web React avec backend Node.js
- Base de données Supabase
- Authentification OAuth
- Paiements Stripe
- Support bilingue FR/EN

## Positionnement
"L'infrastructure humaine de la vie partagée" - Une application qui simplifie la gestion de colocation avec bienveillance.
